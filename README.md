# colorprint

Tired of your terminal projects looking meh???

**WANT TO PRINT COLORFULLY?**

Look no further — colorprint has your back!

It offers:

* Colors
* Styles
* Rainbow prints
* Bold/underline support
* Fun, eye-catching terminal output!

## Installation

```bash
pip install git+https://github.com/CoolGuy158-Git/colorprint.git
```

## Usage

```python
import colorprint

# Basic color
colorprint.cprint("Hello world!", "RED")

# Bold text
colorprint.cprint("Bold text!", "GREEN", style="BOLD")

# Underline text
colorprint.cprint("Underline me", "CYAN", style="UNDERLINE")
```
